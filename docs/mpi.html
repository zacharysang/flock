    mpi

    This module is to be imported by either a browser WebWorker or a nodejs worker-thread If imported by a WebWorker, it will simply be executed. For this reason `mpi` is in the global scope so that it is available after being imported. On the other hand, if imported by a worker-thread, it will be imported using nodejs' module system (using module.exports)
